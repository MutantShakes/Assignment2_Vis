Parallel Coordinate Plot (PCP)

Compressed Natural Gas (CNG) Information

    # Dimensions of Parallel Coordinate Plot :

  1. CNG Pressure (in Pounds per Square Inch)
  2. Number of CNG Dispensers on stations
  3. CNG Vehicle class
    • 1 - Low Duty
    • 2 - Medium Duty
    • 3 - High Duty
  4. Log value of CNG Total Compression Capacity (in Cubic meters per hour)
  5. Log value of CNG Storage Capacity of stations (in Liters)
